\StartSection{Orientation}

If you are new to the project, we are delighted to have you with us. You will probably enjoy working with our community. To help you get started, the following is a simple checklist.

\usesymbols[mvs]
\definesymbol[1][{\symbol[martinvogel 2][CheckedBox]}]
\startitemize[1]
\item
Read the handbook.

\item
Join the announcement mailing list as described in \in{section}[Mailing Lists].

\item
Join the Launchpad \href{https://launchpad.net/~avaneya/+join}{crew}.

\item
Read the Avaneya Code of Conduct and digitally sign it as described in \in{section}[Avaneya Code of Conduct] and \in{section}[Signing the Code of Conduct] respectively.

\item
Join the private mailing list described in \in{section}[Private Discussion]. Remember to introduce yourself.

\item
Check out the branch using the method described in \in{section}[Using Bzr].
\stopitemize
